# 香港徒步路线地图

## 项目简介

这是一个交互式香港徒步路线可视化网页，展示20条适合新手的徒步路线。每个路线都标注了具体位置和特色介绍，用户可以通过地图直观地查看路线分布和详细信息。

## 功能特点

- **交互式地图**：使用Leaflet.js实现，中心点设在香港
- **路线标记**：20个徒步路线位置标记，点击可查看详情
- **路线列表**：左侧边栏列出所有路线，包含简短介绍
- **详情弹窗**：点击路线可查看完整的位置和特色信息
- **响应式设计**：适配桌面和移动设备
- **瓦片服务优化**：自动切换多个地图瓦片服务，确保稳定显示

## 技术栈

- **前端框架**：原生HTML/CSS/JavaScript
- **地图库**：Leaflet.js
- **地图服务**：多瓦片服务切换（OpenStreetMap France、CartoDB、Stamen Terrain）
- **样式**：响应式CSS设计

## 使用说明

1. 打开 `index.html` 文件
2. 点击左侧路线列表中的任意路线，地图会自动定位到对应位置
3. 点击地图上的标记，查看详细的路线信息
4. 在移动设备上，可使用左上角的菜单按钮切换侧边栏
