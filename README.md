## 🌟 项目概述

**香港徒步路线大全**是一个专注香港徒步路线信息的Web应用，为徒步爱好者提供：
- 📍 **丰富的路线数据库** - 涵盖香港各区精选徒步路线
- 🎯 **科学的难度评估** - 基于距离、爬升、地形的5级难度系统
- 🏆 **成就激励系统** - 完成路线获得专属徽章
- 👥 **个人进度追踪** - 记录您的徒步历程和成就
- 📱 **响应式设计** - 适配桌面、平板、手机多端体验

### 📋 核心功能

| 功能模块 | 描述 | 状态 |
|---------|------|------|
| 🗺️ 路线浏览 | 分类筛选、难度排序、详情查看 | ✅ 已完成 |
| 👤 用户系统 | 注册登录、个人资料、进度管理 | 🚧 开发中 |
| 🏅 成就系统 | 路线完成徽章、排行榜、统计 | 🚧 开发中 |
| 📊 数据分析 | 个人统计、路线推荐、趋势分析 | 📋 规划中 |

## 🚀 快速开始

### 环境要求
- Node.js 16+ 
- npm 8+
- 现代浏览器 (Chrome 80+, Firefox 75+, Safari 13+)
- 推荐屏幕分辨率: 1280x720 或更高

### 项目启动

#### 前端开发服务器
```bash
# 1. 进入项目目录
cd "hiking- Hong Kong"

# 2. 安装前端依赖
npm install

# 3. 启动前端开发服务器
npm run dev

# 4. 打开浏览器访问
# http://localhost:5173/pages/index.html
```

#### 后端API服务器
```bash
# 1. 进入后端目录
cd backend

# 2. 安装后端依赖
npm install

# 3. 启动后端服务器
node server.js

# 后端服务运行在 http://localhost:3000
```

### 🎯 在线预览

- **前端应用** → http://localhost:5173/pages/index.html
- **后端API** → http://localhost:3000/api
- **开发服务器**: 自动热重载，修改即更新

### ✨ 新功能体验

🎨 **徒步主题界面**: 
- 移动鼠标查看3D视差效果
- 点击Banner创建足迹特效
- 观察山峰、云层、星空动画

🔍 **路线筛选**:
- 按难度等级筛选 (1-5级)
- 按地区分类浏览
- 查看详细路线信息

### 核心页面预览

| 🏠 **主页** | 徒步主题界面、路线总览、筛选分类 | ✅ 已完成 |

### 🎯 特色功能

- **🌲 徒步主题界面**: 全新山绿色和蓝天白云配色，沉浸式自然体验
  - 🏔️ **多层山峰轮廓**: 动态山脉剪影带滚动动画
  - ☁️ **云层飘动**: 3朵白云不同时间延迟飘动效果
  - ⭐ **星空闪烁**: 20颗随机分布星星闪烁效果
  - 🛤️ **路径线条**: 徒步路线流动动画
  - ☀️ **太阳月亮**: 昼夜循环动画
  - 🚶 **徒步人物**: 行走剪影动画
  - 🖱️ **鼠标视差**: 移动鼠标时特效元素产生3D视差
  - 🐾 **点击足迹**: 点击Banner留下棕色足迹3秒渐隐
  - 🍃 **绿色叶子粒子**: 替代白色粒子，使用香港山景绿色系

- **🔍 智能筛选**: 按难度、距离、标签多维度筛选
- **📊 难度评估**: 基于距离、爬升、地形的5级科学评级系统
- **📊 统计面板**: 个人徒步数据可视化
- **🗺️ 路线管理**: 完整的路线数据库和验证系统
- **🎭 粒子特效**: 旋转钻石形状、发光阴影、20秒更新周期
